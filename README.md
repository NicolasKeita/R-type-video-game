# R-Type
R-Type is a "shoot them up" 2D video game.

Our video game has been created with the SFML library.

## Prerequisites

Conan (to install Boost.asio) : https://conan.io

## Usage

Linux :

``` $ bash build.bash```
 
 ```$ ./r-type_client```
