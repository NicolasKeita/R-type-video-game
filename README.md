# R-Type
R-Type is a "shoot them up" 2D video game.
Our video game has been created with the SFML library.

## Prerequisites

Conan (to install Boost.asio)

## Installation

Linux :

``` bash build.bash```
 
 ```./R_type_video_game```
 

Windows :

``` ask my team ```


## Usage

```bash
start server:
./r-type_server

start client;
./r-type_client
```
